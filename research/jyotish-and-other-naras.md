# Jyotish, Human Design, and the other Naras (seed — Jul 2 2026)

The builder's framing, near-verbatim: *"we will probably need to overlay the tropical astrology
with Jyotish… Maybe Human Design is a synthesis, maybe it is a hallucination. All of those are
Nara trying to make sense of the skies."*

That last sentence IS the project thesis, and it dissolves the usual turf war. Nara doesn't have
to pick which system is "true" — every system is a **lens row**, a historically-situated human
attempt to read the same sky:

| Lens | Sky it reads | What's honest about it | What to say plainly |
|------|--------------|------------------------|---------------------|
| Tropical (this repo's spine) | The SEASONS — solstices/equinoxes, Sun-Earth geometry | Astronomically real cycle; the signs are season-names, not constellation-positions | Precession means tropical Aries ≠ the constellation Aries — and that's fine, it never claimed to be |
| Jyotish (sidereal) | The CONSTELLATIONS — star-referenced zodiac via an ayanāṁśa offset | Tracks the actual stellar backdrop; ancient continuous practice (nakṣatras are its native layer, arguably more than the 12 signs) | The ayanāṁśa (which offset) is itself contested — sidereal isn't one system but a family |
| Human Design (1987, Ra Uru Hu) | A SYNTHESIS claim — astrology + I Ching + kabbalah + chakras | Interesting as a *modern* meaning-making artifact — proof the impulse is alive | Origin is a claimed revelation, decades not centuries of practice; present it as "a recent synthesis," never as ancient lineage |

## Design consequences (when this lane opens)

- **Overlay, don't switch.** The same chart moment (from `calculate-chart.py`'s real sky
  positions) can render with a tropical ring AND a sidereal ring — the ~24° gap between them
  becomes VISIBLE, which is the single most honest teaching device in astrology. The
  disagreement is the content.
- **Each lens is an owned lens** (DESIGN.md rule): "tropical, as we read it," "Jyotish, as
  taught by X tradition, summarized" — summary + hyperlink to real sources, no invented
  authority. For Jyotish especially: link primary-adjacent sources, don't paraphrase doctrine
  as ours.
- **Human Design gets the same humility test the tarot course applies**: report what it claims,
  when it appeared, who made it, and let the reader hold synthesis-or-hallucination themselves.
  Kant-by-mode applies (game / mirror / divination).
- **Grammar shape when built:** one grammar per lens (not one mega-grammar), cross-linked with
  pill links like kali↔paradevi's `emanation` — "this sign in the tropical lens ↔ this
  nakṣatra/rāśi region in the sidereal lens."

Not a trip task — a research lane. Trip-sized first step: collect 3–5 public-domain or
clearly-licensed Jyotish source links (nakṣatra lists, classical texts in translation) into
this file via the GitHub connector.
