# Your First Grammar

This is the chapter where something comes alive. The loop has four moves — research, build, look, refine — and you drive all of them by saying things. Below is a real session shape, with the actual words.

Start a fresh chat at [claude.ai](https://claude.ai). One useful opening habit:

> "Read the README in my GitHub repo `<your-username>/<your-repo>` so you know what this project is."

## Move 1 — research

Ask Claude to gather honest ground before building. It has a Wikipedia tool and a Wikimedia Commons image search through the recursive.eco connector:

> "I want a small grammar of the eight I Ching trigrams. Look up each trigram on Wikipedia and give me, for each: its name, its image in nature, and one sentence I could verify. Cite where each fact came from."

Read what comes back. Push back where it is vague — "which translation says that?" is always a fair question. This is your project's honesty being built in.

## Move 2 — build

When the material feels right:

> "Create a new grammar on recursive.eco called 'The Eight Trigrams'. Description: a study set of the I Ching trigrams, facts and reflections kept separate. Keep it private for now."

Then the items. **One thing to know first: items keep the order you add them in, and there is no reordering later — so decide the order before adding.**

> "Now add all eight trigrams as items, in the traditional order. Each item gets two sections: 'The Source' with the verifiable facts we researched, and 'The Mirror' with one reflective question of ours. Show me the full list in order before you add them, so I can confirm."

Confirming the list first costs one message and saves rebuilding.

## Move 3 — look

Open [flow.recursive.eco](https://flow.recursive.eco) on your phone, go to **Library → My Grammars** (or ask Claude for the direct link to the grammar it just created). Read it as a stranger would. This step is not optional — the app is where the truth is, not the chat.

## Move 4 — refine

Back in the chat, say what you saw:

> "Item 3's mirror question is flat. Rewrite it as something a person could actually sit with."

> "Search Wikimedia Commons for a public-domain image for each trigram — classical Chinese sources preferred — and set them as the item images. Tell me the license and credit for each one you pick."

> "The image on item 5 is wrong, find another."

For subjects with no good historical imagery, Claude can also generate images — but for a study project, real public-domain sources with credits are usually the better spine.

Loop moves 3 and 4 until it feels like yours. That is the whole method. It is the same loop the Trika grammars were built with, from a phone.

---

**Do it now:** run one full loop — research, build a grammar with at least three items, look at it live in the app, and change one thing you did not like.
