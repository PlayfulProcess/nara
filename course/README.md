# From Zero to Your First Altar

A short course for a complete beginner — no coding, phone only — on going from nothing to a living grammar on [recursive.eco](https://flow.recursive.eco), using natural language plus exactly two manual setup steps.

This repo (nara) was itself created by following this loop; see `../journal/2026-07-02.md`.
